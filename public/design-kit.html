Best Practices

Guidelines for using the design system correctly.

Semantic Tokens
DO
bg-surface-base
text-content-primary
border-stroke

Let semantic tokens handle light/dark mode automatically.
DON'T
bg-white dark:bg-gray-900
text-gray-900 dark:text-white

Don't override semantic tokens with explicit dark: variants.
Main Content Area
DO
bg-neutral-100 dark:bg-surface-raised

Use neutral-100 in light mode for softer background. Cards on top appear elevated.
DON'T
bg-surface-base for main content

Pure white is harsh. Use subtle gray for better contrast with card elevation.
Interactive Colors
DO
hover:bg-main-50 dark:hover:bg-main-950

Use dark: variants for interactive states (hover, focus, active).
DON'T
hover:bg-blue-50 (no dark variant)

Always provide dark mode hover states for colored interactions.
